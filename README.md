# Dynamic-NFT-For-Gaming
Dynamic NFT is the use case of Gaming.

## To Build the Application
```shell
>>npm init
// to install revise-sdk
>>npm install revise-sdk --save
// to execute the index.js file
>>node index.js
// to execute the properties.js file
>>node properties.js
```
